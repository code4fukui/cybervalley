# CyberValley

サイバーセキュリティやIoTをはじめとする先端技術に関する交流・活動を行う地域コミュニティ「CyberValley, Japan」のリポジトリです。

## 機能
- CyberFridayの開催
- 地元企業や大学とのコラボイベント開催
- 最新技術情報の共有

## 必要環境
特になし
